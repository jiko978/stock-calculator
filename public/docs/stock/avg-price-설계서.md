# 평균 단가 계산기 (avg-price) 설계서

## 1. 개요
주식 매수 시, 여러 차례에 걸쳐 다른 가격에 매수했을 때의 **최종 평균 단가 및 총 매수 금액**을 계산하는 메뉴입니다. (물타기/불타기 계산)

## 2. 화면 위치
- **경로**: `/stock/avg-price`
- **파일**: `app/stock/avg-price/page.tsx` (서버), `app/stock/avg-price/AvgPrice.tsx` (클라이언트)

## 3. 화면 UI/UX 구성
1. **입력폼 (리스트 형태)**
   - 차수별 입력 행 (1차 매수, 2차 매수, ... 최대 10차수)
   - 입력 항목: 매수 단가 (원), 수량 (주)
   - 모바일/PC 반응형 테이블 적용: 모바일에서는 각 컬럼이 최소 너비(90px)를 유지하여 스크롤 발생, PC에서는 여유있는 비율(25%)로 자동 확장.
   - '추가' 버튼: 새로운 차수의 매수 내역 입력 행 추가
   - '삭제' 아이콘: 해당 차수의 매수 내역 삭제 (단, 최소 1개의 행은 존재해야 함)
2. **현재가 입력폼 (옵션)**
   - 현재가를 입력 시 수익률, 수익금 파악 가능
   - Placeholder("현재가를 입력하면...") 문구는 기기 너비(ResizeObserver)에 따라 폰트 사이즈가 동적으로 자동 조절/축소됨. (단, PC화면에서는 고정)
3. **제어 버튼**
   - **계산하기** : 입력된 데이터를 바탕으로 결과 도출
   - **초기화** : 모든 입력 행을 초기 상태(빈 3개 행)로 되돌림
4. **결과 영역**
   - 수익률 및 등급 뱃지 (현재가 입력 시 표시)
   - 총 매수 수량 (주), 매수 합계 금액 (원), 평가 금액 (원)
   - **평균 매수 단가 (원)** (핵심 강조)
   - 차수별 매수 비중 바 그래픽 제공

## 4. 상태(State) 관리
- `type Row = { id: number; price: string; qty: string }`
- `rows: Row[]` : 사용자가 입력한 매수 내역 리스트 (초기값 길이 3개 제공)
- `currentPrice`: string (현재가)
- `calculated` : boolean (계산 버튼 클릭 여부)

## 5. 계산 로직
1. 매수 단가 × 수량 = 해당 차수 매수 금액
2. 총 매수 수량 = 각 차수 수량의 합
3. 총 매수 금액 = 각 차수 매수 금액의 합
4. **평균 단가 = 총 매수 금액 ÷ 총 매수 수량**

## 6. 예외 처리 가이드 (Validation)
- 매수 단가, 수량, 현재가 모두 문자열/음수 입력 문자 방지 (숫자만 허용, 정규식으로 필터링)
- **최대 입력 길이 제한**: 매수가, 수량, 현재가 모두 **9자리(10억 미만)** 까지만 입력 가능하도록 차단.
- 빈 값이 포함된 경우 필터링하거나 0으로 계산에서 제외.
- 총 수량이 0일 경우, 평균 단가를 Number.NaN 또는 무한대로 처리하지 않고 `0`으로 일괄 처리.

## 7. 스타일(디자인) 참고
- 프로젝트 공통 CSS (tailwind) 및 JIKO-calculator.md 디자인 패턴 준수.
- 입력 시 숫자 길이에 비례해 폰트 사이즈가 동적으로 줄어드는 로직 포함. (PC에서는 14px 고정, 모바일에서만 작동)
- 결과를 클립보드에 복사할 수 있는 기능 제공.
