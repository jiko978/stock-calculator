# bmr-기능명세서.md

## 1. 개요
기초대사량(BMR) 계산 메뉴의 기능 명세서입니다.

## 2. 화면 위치
1. 경로 : `/calculator/health/bmr`
2. 동적 경로 (SEO) : `/calculator/health/bmr/[slug]` (종목별 전용 페이지)
3. 파일
ㄴ 서버 : `app/calculator/health/bmr/page.tsx`
ㄴ 동적 서버 : `app/calculator/health/bmr/[slug]/page.tsx` (활동량별/성별 슬러그 지원)
ㄴ 클라이언트 : `app/calculator/health/bmr/Bmr.tsx`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 건강 계산기 > 기초대사량(BMR) 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(성별, 나이, 키, 체중)에 따라 Title, Description, Keywords 자동 생성
3. **PWA 연동**: 
   - `PWAInstallProvider`를 통한 설치 가능 상태 감지
   - `InstallBanner` (페이지 하단), `Footer` (앱 설치 버튼) 연동
4. 뒤로가기 버튼 / 공유 버튼 (NavBar 컴포넌트 필수 사용)
5. JIKO 플랫폼 홈(`jiko.kr`) 허브 이동 버튼 (Header 및 Footer 영역에 단축 위치)

## 4. 화면 기능
1. 입력 데이터
ㄴ 나이
ㄴ 키(cm)
ㄴ 체중(kg)
ㄴ (선택) 활동 지수 (전혀 활동 없음 ~ 매우 활동적)
ㄴ (선택) 체지방률 (%) - 보다 정확한 Katch-McArdle 공식을 위해 선택 입력 가능

2. 출력 데이터
ㄴ 기초대사량(BMR) - 다양한 공식(Mifflin-St Jeor 등) 기준 제공
ㄴ 활동대사량(TDEE) - 하루 총 에너지 소비량 (활동 지수 선택 시)
ㄴ 나이 대비 기초대사량 순위/분포 정보
ㄴ 하루 권장 섭취 칼로리 및 영양소 가이드 (체중 유지/감량/증량 목적별)
ㄴ **시각화 요소**: 대사량 분포를 통한 "내 몸의 엔진" 상태 이미지
ㄴ 결과 복사 기능 (텍스트 클립보드 복사)

3. 버튼 기능
ㄴ 초기화 : 모든 입력 데이터 삭제
ㄴ 계산하기 : 계산 결과 출력
ㄴ 결과 복사 : 결과 결과 복사

4. UI/UX 기능
ㄴ 초기화 버튼 클릭 시 흔들림 애니메이션 (shake) 효과 적용

5. 하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)
ㄴ 검색 엔진 최적화(SEO)를 위한 JSON-LD 라우팅 보강
ㄴㄴ `SoftwareApplication` 및 `FAQPage` 외 추가 범용 `schema` 스크립트 작성 및 로드
ㄴㄴ 구글 서치 콘솔(GSC) 등록 완료 (루트 경로 HTML 소유권 확인)
ㄴㄴ `sitemap.ts` 및 `robots.ts`를 통한 동적 크롤링 및 인덱싱 허용 (검색 노출 최적화)
ㄴ UI 디자인 : 본체와 동일한 카드형 레이아웃 및 다크모드 대응 적용
ㄴ 구성요소
ㄴㄴ 소개 : 기초대사량(BMR) 및 하루 총 에너지 소비량(TDEE) 계산기 목적 안내
ㄴㄴ 사용 방법 : 성별, 나이, 키, 체중, 활동량을 통한 대사량 계산 가이드
ㄴㄴ 계산 예시 : 30세 남성, 175cm, 70kg 기준 BMR 약 1,600kcal 도출 예제
ㄴㄴ FAQ : BMR 공식(Mifflin-St Jeor 등) 및 나이에 따른 기초대사량 변화 안내 (**공통 `FAQ` 컴포넌트 적용**)
ㄴㄴ 건강 계산기 더 보기 : 다른 건강 계산기 메뉴 바로가기 (**공통 `HealthMoreCalculators` 컴포넌트 적용**)

## 5. 계산 로직
1. 기초대사량(BMR) - Mifflin-St Jeor 공식 (표준):
   - 남성: (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) + 5
   - 여성: (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) - 161
2. 활동대사량(TDEE) = BMR × 활동 지수
   - 주로 앉아서 일함: 1.2
   - 가벼운 활동(주 1~3일): 1.375
   - 보통 활동(주 3~5일): 1.55
   - 활동적(주 6~7일): 1.725
   - 매우 활동적: 1.9

## 6. 예외 처리
1. 입력값 검증 (정규식 필터링)
ㄴ 성별, 나이, 키, 체중은 숫자만 입력 가능 (문자 등 입력 시 무시)
ㄴ 성별, 나이, 키, 체중은 음수 입력 불가능
2. 계산 결과 검증
ㄴ 기초대사량은 숫자만 출력
