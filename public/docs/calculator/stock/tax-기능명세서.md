# tax-기능명세서.md

## 1. 개요
국내 및 해외 주식 매매 시 발생하는 세금(거래세, 양도세)과 각종 수수료를 계산하여 실제 순이익을 도출합니다.

## 2. 화면 위치
1. 경로 : `/calculator/stock/tax`
2. 동적 경로 (SEO) : `/calculator/stock/tax/[slug]`
3. 파일
   ㄴ 서버 : `app/calculator/stock/tax/page.tsx`
   ㄴ 동적 서버 : `app/calculator/stock/tax/[slug]/page.tsx`
   ㄴ 클라이언트 : `app/calculator/stock/tax/StockTax.tsx`

## 3. 화면 기능
### 3.1 입력 데이터
1. **거래 시장**: 코스피, 코스닥, 해외(미국 등) 토글 선택
2. **매수가/매도가/수량**: 기본 거래 정보
3. **수수료(옵션)**: 매수/매도 시 증권사 수수료 (기본값 제공 후 수정 가능)
4. **기본 공제(해외 전용)**: 연간 기본 공제 250만 원 반영 여부

### 3.2 출력 데이터 (결과)
1. **세전 손익**: 수수료/세금 차감 전 이익
2. **세금 상세**:
   - **거래세**: 국내 주식 매도 시 발생하는 세금
   - **양도세**: 해외 주식 수익금(공제 후)의 22%
3. **세후 순수익**: 모든 비용을 제외한 최종 통장에 찍히는 돈
4. **최소 익절가 안내**: 수수료와 세금을 계산했을 때 손해가 나지 않는 최소 매도 가격

## 4. UI/UX 차별화 포인트
1. **손절/익절 컬러 가이드**: 세후 수익이 0원 이하일 경우 테두리 및 텍스트 컬러를 즉각 변경하여 직관적 파악 가능.
2. **시장별 자동 세팅**: 해외 선택 시 환율 입력창이 나타나거나, 최근 환율 기반 가이드라인 제공(추후).

## 5. 계산 로직
1. **국내 세금** = 매도금액 × 거래세율 (코스피 0.03% + 농특세 0.15% 또는 코스닥 0.18% 등 최신 세율 반영)
2. **해외 양도세** = (매매손익 - 250만 원) × 0.22 (단, 손익이 250만 원 미만일 경우 0원)
3. **최종 세후 수익** = 세전 손익 - (매수수수료 + 매도수수료 + 세금)
