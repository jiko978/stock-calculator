# fee-기능명세서.md

## 1. 개요
국내 및 해외 주식 매매 시 발생하는 각종 수수료와 세금(거래세, 양도세)을 계산하여 실제 순이익을 도출합니다.

## 2. 화면 위치
1. 경로 : `/calculator/stock/fee`
2. 동적 경로 (SEO) : `/calculator/stock/fee/[slug]`
3. 파일
   ㄴ 서버 : `app/calculator/stock/fee/page.tsx`
   ㄴ 동적 서버 : `app/calculator/stock/fee/[slug]/page.tsx`
   ㄴ 클라이언트 : `app/calculator/stock/fee/StockFee.tsx`

## 4. 화면 기능
### 3.1 입력 데이터
1. **거래 시장**: 코스피, 코스닥, 해외(미국 등) 토글 선택
2. **매수가/매도가/수량**: 기본 거래 정보
3. **수수료(옵션)**: 매수/매도 시 증권사 수수료 (기본값 제공 후 수정 가능)
4. **기본 공제(해외 전용)**: 연간 기본 공제 250만 원 반영 여부

### 3.2 출력 데이터 (결과)
1. **세전 손익**: 수수료/세금 차감 전 이익
2. **세금 상세**:
   - **거래세**: 국내 주식 매도 시 발생하는 세금 (0.18%)
   - **양도세**: 해외 주식 수익금(공제 후)의 22%
3. **세후 순수익**: 모든 비용을 제외한 최종 통장에 찍히는 돈
4. **최소 익절가 안내**: 수수료와 세금을 계산했을 때 손해가 나지 않는 최소 매도 가격

## 4. 화면 및 디자인 패턴 (Standard Content)
1. **[공통 카드세션]**
   - **메뉴 설명**: 주식 거래 시 발생하는 수수료/세금의 중요성 및 손익분기점 안내
   - **사용 방법**: 거래 시장 선택 및 매매 정보 입력 가이드 (2열 그리드)
   - **계산 예시**: 100만원 매도 시 발생하는 거래세 및 수수료 차감 실례 (2열 그리드)
   - **자주 묻는 질문(FAQ)**: 최소 익절가의 의의 및 해외 주식 양도세 전략 안내 (**공통 `FAQ` 컴포넌트 적용**)
   - **주식 계산기 더 보기**: 다른 주식 계산기 메뉴 바로가기 (**공통 `StockMoreCalculators` 컴포넌트 적용**)
2. **[개별 카드세션]**
   - **세금 상식**: 국내 주식(거래세) vs 해외 주식(양도세) 비교 가이드

## 5. UI/UX 차별화 포인트
1. **손절/익절 컬러 가이드**: 세후 수익이 0원 이하일 경우 테두리 및 텍스트 컬러를 즉각 변경하여 직관적 파악 가능.
2. **시장별 자동 세팅**: 해외 선택 시 환율 입력창이 나타나거나, 최근 환율 기반 가이드라인 제공.

## 6. 계산 로직
1. **국내 세금** = 매도금액 × 거래세율 (코스피 0.18%, 코스닥 0.18% 등 최신 세율 반영)
2. **해외 양도세** = (매매손익 - 250만 원) × 0.22 (단, 손익이 250만 원 미만일 경우 0원)
3. **최종 세후 수익** = 세전 손익 - (매수수수료 + 매도수수료 + 세금)
4. **최소 익절가** = 수수료와 세금을 모두 공제하고도 순수익이 (+)가 되는 첫 번째 매도 가격
