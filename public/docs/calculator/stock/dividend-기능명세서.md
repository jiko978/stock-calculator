# dividend-기능명세서.md

## 1. 개요
주식 배당금 및 배당수익률을 계산하고, 목표 배당금 달성을 위한 투자 계획을 세워주는 계산기입니다.

## 2. 화면 위치
1. 경로 : `/calculator/stock/dividend`
2. 동적 경로 (SEO) : `/calculator/stock/dividend/[slug]`
3. 파일
   ㄴ 서버 : `app/calculator/stock/dividend/page.tsx`
   ㄴ 동적 서버 : `app/calculator/stock/dividend/[slug]/page.tsx`
   ㄴ 클라이언트 : `app/calculator/stock/dividend/Dividend.tsx`

## 3. 화면 기능
### 3.1 입력 데이터
1. **매수가(원)**: 나의 평균 매입 단가
2. **현재가(원)**: 현재 주식 가격 (수수료 및 시가 배당률 계산용)
3. **주당 배당금(원)**: 1주당 지급되는 배당 금액
4. **배당 주기**: 년, 반기, 분기, 월 중 선택
5. **수량(주)**: 현재 보유 중인 주식 수량
6. **목표 월 배당금(옵션)**: 내가 매달 받고 싶은 배당액 (예: 100만 원)

### 3.2 출력 데이터 (결과)
1. **배당 수익**: 주기별(년/월) 세전 및 세후 실수령액
2. **배당 수익률**: 
   - **현재 수익률**: 현재가 기준 배당률
   - **나의 수익률(YoC)**: 실제 매수가 기준 배당률
3. **목표 달성 가이드**:
   - 목표액 달성률 (%)
   - 추가로 필요한 투자 자금 (현재가 기준)
   - 추가로 사야 할 주식 수량

## 4. UI/UX 차별화 포인트
1. **생활 기준 결과**: "이 배당금으로 매달 [아이패드] 한 대를 살 수 있어요!", "한 달 통신비 해결!" 등 친숙한 비유 노출.
2. **결과 강조 UI**: 세후 실수령액을 가장 크게 강조.
3. **고배당주 리스트**: 페이지 하단에 국내/해외 주요 고배당주 및 배당성장주 리스트 카드 배치.

## 5. 계산 로직
1. **총 배당금(세전)** = 주당 배당금 × 수량
2. **세후 총액** = 총 배당금 × (1 - 0.154) (국내 배당소득세 기준)
3. **나의 배당 수익률(YoC)** = (주당 배당금 × 배당 횟수 / 매수가) × 100
4. **추가 필요 수량** = (목표 월 배당금 × 12 / 세후 주당 연 배당금) - 현재 수량
