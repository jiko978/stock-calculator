# loans-기능명세서.md

## 1. 개요
대출 이자 및 월 상환액을 계산하는 메뉴입니다. 다양한 상환 방식에 따른 비용 차이를 비교할 수 있습니다.

## 2. 화면 위치
1. 경로 : `/calculator/finance/loans`
2. 동적 경로 (SEO) : `/calculator/finance/loans/[slug]`
3. 파일
   ㄴ 서버 : `app/calculator/finance/loans/page.tsx`
   ㄴ 클라이언트 : `app/calculator/finance/loans/Loans.tsx`

## 3. 화면 기능
1. **입력 데이터**
   - 대출 금액(원): (단위 버튼: 100만, 1000만, 5000만, 1억 등)
   - 대출 기간(개월/년)
   - 연 이자율(%)
   - 상환 방식: 원리금균등, 원금균등, 만기일시상환
   - 거치 기간(옵션): 이자만 내는 기간 설정

2. **출력 데이터**
   - 총 상환 금액
   - 총 대출 이자
   - 1회차 상환액 (또는 평균 월 상환액)
   - 회차별 상환 상세 (표/그래프): 원금, 이자, 잔액 변화

3. **UI/UX 기능**
   - 상단 종목 배지: `💰 대출 이자 계산기`
   - 상환 스케줄 차트: 기간 흐름에 따른 잔액 감소 시각화

## 4. 하단 콘텐츠 (Standard)
1. 소개: 대출 상환 방식에 따른 이자 비용 차이 안내
2. 사용 방법: 거치 기간 및 상환 방식 선택 가이드
3. 계산 예시: 1억 대출, 3.5%, 30년 상환 초기 월 납입액 예시
4. FAQ: 원리금균등 vs 원금균등 차이점 및 중도상환수수료 개념
5. 금융 계산기 더 보기: `FinanceMoreCalculators`

## 5. 계산 로직 (원리금균등 상환 예시)
- **월 상환액** = [대출원금 × 월금리 × (1 + 월금리)^기간] / [(1 + 월금리)^기간 - 1]
  *(월금리 = 연이율 / 12 / 100)*
