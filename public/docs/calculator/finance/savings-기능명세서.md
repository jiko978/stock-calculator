# savings-기능명세서.md

## 1. 개요
적금 이자 계산 메뉴의 기능 명세서입니다. 매월 일정액을 저축하여 만기 시 수령액을 도출합니다.

## 2. 화면 위치
1. 경로 : `/calculator/finance/savings`
2. 동적 경로 (SEO) : `/calculator/finance/savings/[slug]`
3. 파일
   ㄴ 서버 : `app/calculator/finance/savings/page.tsx`
   ㄴ 클라이언트 : `app/calculator/finance/savings/Savings.tsx`

## 3. 화면 기능
1. **입력 데이터**
   - 월 적립액(원): 매달 넣는 금액 (단위 버튼: 1만, 5만, 10만, 50만 등)
   - 적립 기간(개월/년): 6, 12, 24, 36개월 등
   - 연 이자율(%): 소수점 2자리
   - 이자 계산 방식: 단리, 월복리
   - 과세 유형: 일반과세(15.4%), 비과세, 세금우대

2. **출력 데이터**
   - 세후 수령액 (가장 크게 강조)
   - 원금 합계
   - 세전 이자
   - 이자소득세 (세율 표시)
   - 세후 이자

3. **UI/UX 기능**
   - 상단 종목 배지: `🏦 적금 계산기`
   - 금액별 시각화: 원금과 이자 비중을 보여주는 바 차트

## 4. 하단 콘텐츠 (Standard)
1. 소개: 목돈 마련을 위한 적금의 중요성
2. 사용 방법: 월 납입액 및 기간 설정 가이드
3. 계산 예시: 월 100만원, 1년, 5% 적금 시 이자 계산 실례
4. FAQ: 적금 이자가 예금 이자보다 적게 느껴지는 이유 설명 (일할 계산 방식)
5. 금융 계산기 더 보기: `FinanceMoreCalculators`

## 5. 계산 로직 (단리 기준)
- **월별 이자** = 월 적립액 × 이율/12 × (남은 개월 수 / 12)
- **총 세전 이자** = 모든 월별 이자의 합계
- **세후 이자** = 총 세전 이자 × (1 - 과세율)
- **만기 수령액** = (월 적립액 × 기간) + 세후 이자
